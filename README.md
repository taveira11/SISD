# Intelligent Decision Support Systems

Academic project developed for the Intelligent Decision Support Systems course.

## About the project

This project explored different approaches to intelligent decision support, combining symbolic reasoning, machine learning and modern AI techniques.

## Main components

- Developed a rule-based decision support system using Prolog.
- Created a knowledge base with facts, rules and logical inference.
- Implemented decision-making logic inspired by structured triage/recommendation processes.
- Applied Machine Learning using Decision Trees to learn interpretable rules from data.
- Analysed decision paths and feature importance to explain model behaviour.
- Compared automatically learned rules with logic-based rules defined in Prolog.
- Developed a second component using Large Language Models to support interaction, explanation and decision-making.
- Explored the relationship between rule-based systems, explainable machine learning and LLM-based intelligent systems.

## Technologies used

- Prolog
- Python
- Scikit-learn
- Decision Trees
- Machine Learning
- Explainable AI
- Large Language Models
